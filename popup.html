Thanks for using Kraken's IMEI Lookup Extension!

This tool fetches an API URL for quick lookups via an IMEI to result the make/model of a device.

Upon clicking the "check" button, the script will fetch for the results. If the result fails the first time, it will assume CloudFlare or CAPTCHA is active, and open a new tab. You shouldn't have to click off of this extension window.

If the result fails to parse again after the check, or if the request returns invalid, it assumes the IMEI is invalid.

The rate is limited to 30 requests per minute.

Right now, you need CloudFlare verification each time the requests are refreshed (1 minute).
